# Gulp.js Kickstarter
Static website kickstarter using: nunjucks, sass, sprite, jshint & browsersync.

## Usage
Clone this repository and type the commands below:

```terminal
$ npm install
$ gulp build
$ gulp
```
The default task will be watching entire project.
